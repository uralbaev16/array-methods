Array Methods

Massiv metodlari haqida qisqacha ma'lumot va foydalanish usullari

toString()

Tavsifi: Massivni satrga (string) aylantiradi.

const arr = [1, 2, 3];
const result = arr.toString();
console.log(result);
// Natija: "1,2,3"

join()

Tavsifi: Massiv elementlarini biriktirib, satrga aylantiradi.

const arr = [1, 2, 3];
const result = arr.join('-');
console.log(result);
// Natija: "1-2-3"

pop()

Tavsifi: Oxirgi elementni olib tashlaydi va uni qaytaradi.

const arr = [1, 2, 3];
const last = arr.pop();
console.log(last, arr);
// Natija: 3, [1, 2]

push()

Tavsifi: Massivga yangi element qo'shadi va yangi uzunlikni qaytaradi.

const arr = [1, 2];
const newLength = arr.push(3);
console.log(newLength, arr);
// Natija: 3, [1, 2, 3]

shift()

Tavsifi: Birinchi elementni olib tashlaydi va uni qaytaradi.

const arr = [1, 2, 3];
const first = arr.shift();
console.log(first, arr);
// Natija: 1, [2, 3]

unshift()

Tavsifi: Massivning boshiga yangi element qo'shadi.

const arr = [2, 3];
const newLength = arr.unshift(1);
console.log(newLength, arr);
// Natija: 3, [1, 2, 3]

concat()

Tavsifi: Bir nechta massivlarni birlashtiradi.

const arr1 = [1, 2];
const arr2 = [3, 4];
const result = arr1.concat(arr2);
console.log(result);
// Natija: [1, 2, 3, 4]

slice()

Tavsifi: Massivdan ko'rsatilgan qismini qaytaradi.

const arr = [1, 2, 3, 4];
const part = arr.slice(1, 3);
console.log(part);
// Natija: [2, 3]

splice()

Tavsifi: Elementlarni olib tashlaydi yoki qo'shadi.

const arr = [1, 2, 3, 4];
arr.splice(1, 2, 'a', 'b');
console.log(arr);
// Natija: [1, 'a', 'b', 4]

length

Tavsifi: Massiv uzunligini qaytaradi.

const arr = [1, 2, 3];
console.log(arr.length);
// Natija: 3
