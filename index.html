JavaScript Array Methods

Array metodlarini oʻrganing va amaliyotda qoʻllang!

1. map()

map() massivi elementlarini qayta ishlaydi va yangi massiv qaytaradi. 

        
const numbers = [1, 2, 3, 4];
const squares = numbers.map(num => num ** 2);
console.log(squares); // [1, 4, 9, 16]

2. filter()

filter() massivi elementlarini shartga qarab saralaydi va yangi massiv yaratadi. 

        
const numbers = [1, 2, 3, 4, 5];
const evenNumbers = numbers.filter(num => num % 2 === 0);
console.log(evenNumbers); // [2, 4]

3. reduce()

reduce() massiv elementlarini yigʻadi va bitta qiymat qaytaradi. 

        
const numbers = [1, 2, 3, 4];
const sum = numbers.reduce((total, num) => total + num, 0);
console.log(sum); // 10
